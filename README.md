# Boilerplate for new Hoodie App

You can start to build your app in a fingersnap.


## Installation

`$ hoodie new appName -t "zoepage/hoodie-boilerplate" `


## Including

+ jQuery-1.9.1
+ Bootstrap 3

index.html
+ hoodie-accountbar (SignIn / SignOut)
+ including all (css + js) files you need

main.css 

main.js
+ init hoodie
